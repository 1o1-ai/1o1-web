Embed Abhyas Testing on your site

Host the same UI as yogabrata.com/testing on your pages. Partners receive a Bearer token and optional embed key from the Brahmando team.

Option 1 — iframe (simplest)

<iframe
  src="https://yogabrata.com/testing/?embed=1&token=YOUR_TOKEN&subject=Science&count=8"
  title="Abhyas CBSE Class 10"
  style="width:100%;min-height:900px;border:0;border-radius:16px"
  allow="clipboard-write"
></iframe>

Option 2 — script widget

<div id="abhyas-testing"></div>
<script
  src="https://yogabrata.com/testing/abhyas-testing-widget.js"
  data-container="abhyas-testing"
  data-token="YOUR_TOKEN"
  data-embed-key="YOUR_EMBED_KEY"
  data-subject="Science"
  data-count="8"
  data-height="920px">
</script>

Query / data attributes

ParamDescription
embed=1Compact chrome (hides hero/footer; token from URL)
tokenAbhyas Bearer token (required for partner sites)
embed_keyX-Education-Embed-Key for non-allowlisted origins
apiOverride Abhyas API base (default: production)
subject, chapter, countPre-fill practice setup (count 1–40)
modepractice or mock_exam
scopeofficial (default) or anyo

Answer pack with images

When students draw or upload diagrams, answers serialize as JSON strings in the pack:

{"text":"PbI₂ is yellow precipitate","images":[{"type":"draw","name":"diagram.png","data":"data:image/png;base64,..."}]}

After submit, students can Download PDF answer sheet (images embedded) or Download JSON pack (full fidelity for backend ingestion).
